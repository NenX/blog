## #define
使用 `cpp 1.cpp` 可以查看转换后的结果

1. ##：连接

```cpp
#define A(b) _##b##_
int main()
{
int A(1) = 0; // int _1_ = 0;
return 0;
}
```

2. 前加#，将标记转换为字符串．
```cpp

#define A(b) #b
int main()
{
char* a = A(1); // char *a = "1"
return 0;
}
```
综合
```cpp
#include <iostream>
#define paster(n) std::cout<<"token" << #n << "=" << token##n <<std::endl;

int main()
{
int token9 = 10;
paster(9);
return 0;
}
```
