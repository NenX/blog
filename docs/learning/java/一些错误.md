## Java 错误：找不到或无法加载主类（源文件中含有包名 package）
```java
package com.example;

public class Hello{
    public static void main(String[]args){
        System.out.println("Hello");
    }
}
```

```bash
$ javac Hello.java
$ java Hello
错误: 找不到或无法加载主类 Hello
```
解决
```
$ javac com/example/Hello.java
$ java com.example.Hello
```
