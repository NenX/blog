## tar 参数

| 命令  |      英文      |     释义      |
| :---: | :------------: | :-----------: |
|  c/x  | create/extract |   打包/解包   |
|   v   |    verbose     | 显示详细过程  |
|   f   |      file      |     文件      |
|   z   |      gzip      | gzip压缩/解压 |

1. 打包
```sh
tar -cvf files.tar files/ #[file]
```
2. 拆包
```sh
tar -xvf files.tar
```


## gzip 参数
| 命令  |    英文    | 释义  |
| :---: | :--------: | :---: |
|   d   | decompress | 解压  |


1. 压缩

```sh
gzip files.tar
```
2. 解压
```sh
gzip -d files.tar.gz
```
## 合并
1. 打包及压缩
```sh
tar -czvf files.tar.gz files/ #[file]
```
2. 解压及拆包
```sh
tar -xzvf files.tar.gz
```
